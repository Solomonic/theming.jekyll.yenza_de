Introduction into JAM Stack

8 Nov 2019: The JAMstack isn’t a specific set of tools, but a new, modern way of building websites & apps. Any dynamic programming during the request/response cycle is handled by JavaScript, running entirely on the client. All server-side functions or databases actions are abstracted into reusable APIs, accessed over HTTPS with JS. Be it SaaS, third-party services or custom-built. Templated markup should be prebuilt at build time, usually using a site generator for content sites, or a build tool for web apps.
